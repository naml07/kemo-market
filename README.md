# 🏺 KEMO MARKET

> Marketplace Premium Afro-Luxe — Lomé, Togo

## 🎨 Design System
- **Dark mode profond** (#07070a) + **Or africain** (#D4A853)
- Motifs kente en CSS
- Glassmorphism sur les cartes
- Animations fluides
- 100% responsive mobile-first

## 📄 Pages incluses
| Page | Description |
|------|-------------|
| `index.html` | Homepage avec hero, catégories, flash deals, témoignages |
| `catalogue.html` | Catalogue avec filtres, recherche, tri |
| `produit.html` | Fiche produit avec galerie, avis, vendeur |
| `checkout.html` | Commande étape par étape + Mobile Money |
| `vendeur.html` | Landing page vendeurs + formulaire inscription |

## 🚀 Stack Technique
- HTML5 sémantique + ARIA
- CSS3 (variables, Grid, Flexbox, animations)
- JavaScript ES6+ vanilla (zéro dépendance)

## 🔑 Fonctionnalités
- ✅ Panier persistant localStorage
- ✅ Filtres dynamiques par catégorie, prix, note
- ✅ Recherche en temps réel
- ✅ Timer offres flash en direct
- ✅ Cart drawer animé avec overlay
- ✅ Paiement Mobile Money (MTN, Moov, Wave)
- ✅ Formulaire checkout complet
- ✅ Page vendeur avec grille tarifaire
- ✅ Wishlist avec toggle
- ✅ Observer pour animations au scroll
